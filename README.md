Задача
Написать заглушку на Spring используя Java 11, которая принимает get запрос по endpoint /message и отправляет json с рандомным 6-и значным числом с пробелом формата:
{
“id” : “168 159”
}

Принимает post запрос по endpoint /client. Приходит json формата:
{
“id” : “168 159”,
“data” : {
	“name” : “Vasilii”,
	“orgId” : “JK0001”
}
}
В ответе отправляется json с корреляцией формата:
{
	“OK” : true,
	“orgId” : “JK0001”
}

Для отправки запросов и отладки можно использовать Postman.
